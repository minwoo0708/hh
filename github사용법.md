# **github** 
1. github란
   -

 -  git으로 관리하는 프로젝트를 인터넷에 저장하고 공유할 수 있는 사이트

 - 내 컴퓨터에 있는 코드를 인터넷에 저장해서 언제든지 가져올 수 있게 저장하는 공간

 - 여러 사람이 같은 프로젝트를 함께 개발할 때 코드를 쉽게 공유하고 관리할 수 있게 해주는 공간

 - 코드가 수정될 때마다 기록이 남아서 이전 상태로 돌아갈 수 있게 해주는 프로그램


2. 사용하는 이유
    -
- 코드를 인터넷에 저장해서 백업할 수 있기 떄문에

- 여러 사람이 같은 프로젝트를 같이 개발할 수 있게 하기 때문에

- 코드가 수정된 기록을 관리할 수 있기 때문에
