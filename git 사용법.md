# git사용법
1. 핵심 용어
   -
    - ```레지스토리```: 프로젝트들이 모여있는 폴더


    - ```커밋(commit)```: 현재 작업 상태를 저장하는것

    - ```푸쉬(push)```: 내 컴퓨터에 있는 커밋들을 깃허브로 보내는 것

    - ```풀(commit)```: 깃허브에 있는 최신내용을 내 컴퓨터로 가져오는 것 **푸쉬와 반대**

    - ```클론(clone)``` : 깃허브에 있는 프로젝트를 내 컴퓨터로 복사하는 것

    - ```브랜치(branch)``` : 원본 코드에 영향을 주지 않고 따로 작업할수 있는 작업 공간 같은거

2. 터미널 명령어
   -
    - ```git add [파일명]```: 저장된 파일을 목록에 올리는 것(**add.은 모든 파일**)

    - ```git commit -m``` "메세지" : 어떤 작업을 했는지 설명을 적고 저장하는 것

    - ```git restore``` [파일명] : 아직 커밋 안 한 수정 사항을 취소하고 마지막 커밋 상태로 되돌리는 것

    - ```git init``` : 현재 폴더를 git 프로젝트로 만드는 것

    - ```git commit -m "메세지"``` : 어떤 작업을 했는지 설명을 적고 저장하는 것

    - ``` git status``` : 현재 파일 상태를 확인하는 것
